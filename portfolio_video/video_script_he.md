# סרטון הדגמה: Israeli Docs — חילוץ נתונים ממסמכים פיננסיים באמצעות AI

פורמט: 16:9, ‏1080p, כ־80–90 שניות, עברית, קריינות גברית.

המבנה: הוק → בעיה → תהליך אחד מקצה לקצה → בקרת איכות → מקרה קצה → פרטיות → קריאה אחת לפעולה.

## סצנות וקריינות

1. **התוצאה מיד בהתחלה**

   חשבונית בעברית הופכת לנתונים מובנים בתוך שניות — בלי להעתיק ידנית שורות לטבלה.

2. **תהליך אחד ברור**

   מעלים קובץ PDF או תמונה. השירות מזהה גם מסמך רב־עמודי ומחזיר קובץ JSON מובנה אחד.

3. **תוצאה עסקית ברורה**

   בתוצאה אחת מקבלים ספק, לקוח, שורות חיוב, מע״מ, סכום סופי ומספר הקצאה. אפשר להעביר את השדות למערכת הנהלת החשבונות או לבדוק אותם ידנית.

4. **בקרה במקום אמונה עיוורת**

   אחרי החילוץ מופעלת בדיקה חשבונאית עצמאית: סכום השורות, המע״מ והסכום הסופי חייבים להתאים. הבדיקה פועלת גם בלי נתוני אמת להשוואה.

5. **מקרה קצה — הטעויות גלויות**

   טעויות לא מוסתרות. בבדיקה אמיתית שנשמרה, שמונה עשר מתוך עשרים ושניים שדות תאמו. ההבדלים מסומנים, ולכן המסמך עובר לבדיקה במקום להיכנס בשקט למערכת.

6. **נתונים ופרטיות**

   הדמו משתמש רק בנתונים סינתטיים. למסמכי לקוחות יש מצב API מסחרי, והמפתח נשאר בצד השרת. אפשר גם לספק את המערכת בהתקנה מקומית או בתוך הארגון.

7. **קריאה לפעולה**

   שלחו שלושה עד חמישה מסמכים טיפוסיים. אעריך ללא עלות את מבנה השדות, הסיכונים והמתכונת לפיילוט קצר עבור התהליך שלכם.

## עובדות שמוצגות בסרטון

- נתוני הדמו סינתטיים בלבד.
- בבדיקה האמיתית השמורה תאמו 18 מתוך 22 שדות — 81.8%.
- הבדיקה החשבונאית פועלת גם ללא ground-truth.
- פרטים אישיים ופרטי תשלום מוסתרים בסרטון הציבורי.
- מצב API מיועד למסמכי לקוחות; מצב SDK לפיתוח ו־MOCK לבדיקת ממשק ללא רשת.
